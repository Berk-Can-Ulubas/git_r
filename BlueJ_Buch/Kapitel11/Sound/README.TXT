Projekt: Sound - ein einfacher Sound-Player
Autoren: David Barnes und Michael Kölling

Dieses Projekt ist Teil des Zusatzmaterials zum Buch

   Java lernen mit BlueJ - eine Einführung in die 
   objektorientierte Programmierung, 4. Auflage
   David J. Barnes und Michael Kölling
   Pearson Education Deutschland, 2008

Es wird in Kapitel 11 besprochen.

Dieses Projekt implementiert einen einfachen Sound-Player, in erster Linie
für das Experimentieren bei der GUI-Konstruktion.

Zum Starten erzeugen Sie ein Objekt der Klasse SoundPlayerGUI.

Dieser Player kann WAV-, AIFF- und AU-Dateien mit Transferraten von 22.050 KHz
oder 11.025 KHz abspielen. (Dies bedeutet, dass nicht alle Dateien in den
genannten Formaten abgespielt werden können - die jeweilige Kodierung ist
entscheidend.)
